# Excel-Project-1
# Project Title
Data Exploration and analysis

## Description
In this project some of the  basic excel functions,conditional functions,logical functions are used to completely analyse the given data and summarization had been given as follows.It performs sum,average,count,minimum,maximum functions and also IF,SUMIF,COUNTIF functions.Some of the text formatting functions such as RIGHT,LEFT,MID functions are also used for data analysis.

## Getting Started

### Dependencies

Ms-Excel

### Installing
Our project can be downloaded in excel document as well as PDF format


### Executing program

The program can be run in microsoft excel and output had been updated in the document.Its corresponding formulas are in the same document.
